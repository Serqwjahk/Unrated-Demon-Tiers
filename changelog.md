# Changelog

## v1.0.0

- Demon tier picker (Easy to Extreme) for your own unrated demon levels, opened with a configurable key (default **T**).
- "Unrated X Demon" display on the level info page, and updated difficulty icons in level lists.
- Shared tiers: picking one uploads it to the server (secure login via Argon) so every user of the mod sees it. No account needed to view.
- English/Spanish UI.
